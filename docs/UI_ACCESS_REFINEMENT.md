# UI Access Refinement

## Alur Website
Aplikasi Jayusmart digunakan untuk monitoring minimarket berbasis role. Setelah user login, sistem mengecek role, lalu menampilkan menu sesuai hak akses.

## Popup Profil
User dapat melihat informasi nama, role, cabang, akses profile, dan logout melalui popup profil pada navbar.

## Icon Sidebar dan Dashboard
Icon digunakan untuk memperjelas fitur dan mengganti teks initial seperti JS pada card dashboard.

## Foto Produk
Foto produk belum wajib karena sistem ini berfokus pada monitoring transaksi dan stok, bukan e-commerce. Data utama produk adalah kode produk, nama produk, kategori, satuan, harga beli, harga jual, dan stok. Foto produk dapat ditambahkan sebagai fitur tambahan jika waktu mencukupi.
